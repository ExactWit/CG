# 1. 问题定义

**核心问题**：在逆向工程或粗粒度建模中获得的工业网格（通常带噪声、孔洞、非流形边、自相交等缺陷），现有方法多为**分阶段处理**（先修复、再简化、后特征提取），导致误差累积、特征丢失，且无法生成可编辑的层次结构。

**新视角**：能否**端到端**地同时完成“几何修复+特征感知简化+层次化B-rep生成”，输出一个轻量、干净、带特征识别的多层次模型（例如：基面/过渡面/凸台/孔洞的层次树）？

---

# 2. 数据集

可构建一个混合数据集：

- **合成数据**：
    
    - 从[ABC数据集](https://archive.nyu.edu/handle/2451/61215)中选取机械零件CAD模型，导出为B-rep。
        
    - **破坏处理**：在网格上随机注入噪声、删除面片制造孔洞、引入非流形边、自相交，模拟扫描缺陷。
        
    - 保留原始的B-rep结构作为“真实层次”的ground truth。
        
    
- **真实数据**：
    
    - 从[TraceParts](https://www.traceparts.com/)等获取公开的STEP格式零件库，做相同破坏处理。
        
    - 收集少量3D扫描仪获得的真实带缺陷网格（如[Thingi10K](https://ten-thousand-models.appspot.com/)中的破损模型）。
        
    
- **划分**：训练集（800个合成模型+200个真实模型），验证集与测试集各100个。

# 3. 基线方法

- **传统流水线**：
    
    1. MeshLab的筛选、补洞、重网格化。
        
    2. 基于VSA（Variational Shape Approximation）的网格简化与面片划分。
        
    3. 基于拟合（如RANSAC）的几何特征识别。
        
    
- **深度学习方法**：
    
    - [MeshCNN](https://github.com/ranahanocka/MeshCNN/)用于分类分割，但需完整网格。
        
    - [PointTriNet](https://arxiv.org/abs/1908.04014)等补洞网络，但不生成层次。
        
    - [BRepNet](https://arxiv.org/abs/2004.14816)（针对B-rep的分类分割），但需要完整B-rep输入。
        
    
- **对比目标**：你的方法需要在“修复质量”（倒角/孔洞重建精度）与“结构识别率”（特征面正确分割、层次父子关系重建）上超越基线。
    

---

# 4. 目标

- **主要目标**：输入带缺陷的三角网格，输出：
    
    1. **修复后的干净网格**（水密、流形、无自交）。
        
    2. **层次化面片分组**：例如，基座平面→侧壁圆柱面→过渡倒角面→螺纹孔，形成树状依附关系。
        
    3. **轻量化B-rep表达**：每个面片用类型（平面/圆柱/球面等）与参数表示，并包含拓扑连接。
        
    
- **评估指标**：
    
    - 几何误差：Chamfer Distance（修复后 vs GT CAD）。
        
    - 结构误差：面片分类的F1-score、层次关系重建的准确率。
        
    - 效率：处理时间 vs 传统流水线。

# 5. 可能的思考点（研究难点与创新点）

### 5.1 层次结构如何定义与生成？

- **定义**：借鉴CAD的“特征树”，定义父子关系（例如，一个通孔特征依附于某个平面特征）。但需自动推断。
    
- **生成方法**：用GNN迭代聚类网格面片，同时预测每个聚类的几何类型与父特征ID。可设计一个“层次消息传递网络”。
    

### 5.2 如何统一几何修复与结构识别？

- 传统修复会模糊特征边。可考虑**联合优化**：一个双分支网络，一支输出隐式场补全几何，另一支输出特征边场，两者通过可微分泊松重建结合，使补全区域自然延续特征。
    

### 5.3 小样本与泛化问题？

- 合成数据训练，真实数据测试会有域差距。可使用**自监督预训练**：在大量无缺陷模型上，让网络学习“正常”几何规律与特征结构，再在少量缺陷数据上微调。
    

### 5.4 如何实现“可编辑”的输出？

- 不仅输出网格，还可输出程序性参数（如“孔直径=5mm”）。这需要**符号回归**：从重建的几何实体中拟合参数，并与数据库中的标准特征匹配。
