>[!definition]
>**柱面（Cylinder）**
>$\mathbb{R}^3$中到某中轴线距离恒定（非零）的点集构成柱面。给定过$p_i$的中轴线的单位方向向量$\|v_i\|_2^2=1$,半径为$r_i$的柱面$C_i$为如下方程解集：
>$$
>\|v_i\times(x-p_i)\|_2^2=r_i^2,
>$$

两个柱面的相交情况，结论先行：

| 轴线关系         | 半径关系                               | 交线类型     |
| ------------ | ---------------------------------- | -------- |
| 重合           | $r_1=r_2$                          | 柱面       |
|              | $r_1\ne r_2$                       | 空集       |
| 平行(距离$d$)    | $d>r_1+r_2$或$d<\|r_1-r_2\|$(相离)    | 空集       |
|              | $d=r_1+r_2$或$d=\|r_1-r_2\|$（外切/内切） | 1条直线（母线） |
|              | $\|r_1-r_2\|<d<r_1+r_2$            | 2条直线（母线） |
| 相交($\theta$) | $r_1=r_2$                          | 2条椭圆     |
|              | $r_1\ne r_2$                       | 四次曲线     |
| 异面           | $r_1=r_2$                          | 2条椭圆     |
|              | $r_1\ne r_2$                       | 四次曲线     |
>[!remark]
>只要两圆柱半径相等，无论轴线相交还是异面，交线总是两条椭圆；

>[!theorem]
>贝祖定理(Bézout Theorem)
>两个次数分别为$m$和$n$的复射影代数曲面，若它们的交线分解为若干不可约分支，则各分支次数之和（计重数）等于$m\times n$
