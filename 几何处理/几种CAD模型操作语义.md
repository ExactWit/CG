```
Extrude / Revolve   ← 扫掠方式（拉 / 转）
   ├── Side         ← 扫掠方向生成的"侧壁"（柱面/锥面/平面）
   └── End          ← 扫掠两头"封端面"（草图平面位置那两个帽）
Cut / Fillet / Chamfer ← 布尔/过渡操作，不归 Extrude/Revolve 管
```

- **Extrude**​ = 草图沿直线方向拉 → 出来的是棱柱/圆柱
    
- **Revolve**​ = 草图绕轴转 → 出来的是回转体（轴、盘、环）
    
- **Side**​ = 草图轮廓边"扫"出来的面（柱面方向感的那些）
    
- **End**​ = 草图所在平面位置"封"出来的面（两头盖帽）
    
- **Cut**​ = 减材料的版本，几何形态和 Extrude 一样，只是布尔是 subtract
    

## 👁️ 肉眼怎么认——拿典型零件对号

### 1️⃣ ExtrudeSide（拉伸侧壁）

> 长方体：四个竖面都是 ExtrudeSide
> 
> 圆柱：外圆柱面也是 ExtrudeSide（草图是个圆，沿 Z 拉，圆的边扫出柱面）
> 
> 识别特征：**面法向和拉伸方向垂直**，面本身是"沿着拉伸方向长"的——平面/柱面/锥面（如果加了拔模）

### 2️⃣ ExtrudeEnd（拉伸端盖）

> 长方体：上下两个面
> 
> 圆柱：顶面和底面那两个圆饼面
> 
> 识别特征：**面法向和拉伸方向平行**，就是草图原来那张平面的位置，封住扫掠的两头

### 3️⃣ CutSide（切槽/孔的侧壁）

> 在方块上打个通孔 → 孔的内壁四个面 = CutSide
> 
> 沉头孔 → 小孔段内壁是 CutSide，大沉头段内壁也是 CutSide
> 
> 识别特征：**几何形态 = ExtrudeSide，但布尔是"减"**——你看它是"掏空"出来的而不是"长出来"的，就是 Cut 不是 Extrude。最简单判据：这个面如果去掉，零件会变空心 → Cut 系；这个面如果去掉，零件会缺一块实体 → Extrude 系（加材料）。

### 4️⃣ CutEnd（切到头的封端面）

> 盲孔的底部那个面 = CutEnd
> 
> 通孔的话，出口处如果切到外表面形成的"环形端面"也算 CutEnd（有些定义里通孔两端没 End，只有 Blind 才有）
> 
> 识别特征：**Cut 操作里草图平面位置那俩帽**，和 ExtrudeEnd 长得一模一样，区别只在"加还是减"。

### 5️⃣ RevolveSide / RevolveEnd（旋转版的 Side/End）

> 一根轴：外圆柱面 = RevolveSide，两端面 = RevolveEnd
> 
> 识别特征：**所有母线是"圆弧"不是"直线"**——ExtrudeSide 的 U/V 参数是直线扫，RevolveSide 是圆扫。肉眼上回转体（轴、盘、环、球冠）的"侧面"基本都是 RevolveSide，两头封帽是 RevolveEnd。

### 6️⃣ Fillet（圆角）vs Chamfer（倒角）

这两个不属于 Extrude/Revolve/Cut 体系，是**独立的过渡特征**：

|     | Fillet（圆角）          | Chamfer（倒角）                |
| --- | ------------------- | -------------------------- |
| 截面  | **R 恒定**的圆弧         | **C 恒定**的 45° 斜线（或指定距离×角度） |
| 视觉  | 两面交出来的"圆弧过渡"，R 处处相等 | 两面交出来的"斜面"，棱线变一条窄平面        |
| 例子  | 轴肩根部的 R2、内腔尖角倒 R0.5 | 螺栓头部 1×45°、方块棱边 C1         |

肉眼最快区分：**放大看截面，弧线 = Fillet，斜直线 = Chamfer**。

## 🎯 给你一套"看图反推"口诀

拿到一个加工好的零件，想反推某个面属于哪种：

1. **先看法是怎么来的**——是拉出来的（棱线平行）还是转出来的（母线是弧） → 定 Extrude 还是 Revolve
    
2. **再看这个面是"沿着扫掠方向长的"还是"拦在扫掠两头封住的"**​ → 定 Side 还是 End
    
3. **最后看是加材料还是减材料**​ → Extrude/Cut 互转，End/Side 不变
    
4. **圆角/倒角单独拎出来**​ → 不管前后工序，只要是 R 过渡就是 Fillet，C 斜接就是 Chamfer
