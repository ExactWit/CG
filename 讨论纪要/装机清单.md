# 集群配置方案
考虑到课题组已配备多台 RTX 4090D 和 RTX 5880 Ada 工作站，现有工作站受限于单 PCIe x16 插槽，无法继续扩展 GPU 数量，且分散部署不利于统一管理和大规模训练。因此，本项目拟采购一台支持 **8 张双宽 PCIe GPU** 的标准 **4U GPU 服务器平台** 作为集中计算节点，优先采用 **AMD EPYC + PCIe Gen5** 架构。该平台可直接兼容现有 GPU，并支持后续逐步扩充 RTX PRO 系列 GPU，实现"先整合现有资源、后按需扩容"的建设方式。

相较于 HGX/NVSwitch 平台，PCIe GPU 服务器成本更低、兼容消费级与专业级 GPU，更符合当前几何处理、CAD/B-Rep 重建、三维视觉、多模态模型训练等科研任务的需求；未来如需进一步提升算力，还可通过增加 8 GPU 节点并结合高速网络组成多节点 GPU 集群，在保护已有投资的同时具备良好的扩展能力。

为满足多 GPU 并行训练需求，服务器平台需提供充足的 PCIe Gen5 通道、高内存带宽和长期稳定运行能力。AMD EPYC 处理器采用单路即可提供丰富的 PCIe Gen5 通道资源，并支持 12 通道 DDR5 ECC 内存，可满足 8 张 GPU、高速 NVMe 存储及高速网络接口的同时部署，具有良好的扩展能力和性价比。目前主流 8 GPU PCIe AI 服务器（如 Supermicro、Gigabyte、ASUS 等）均广泛采用 AMD EPYC 平台，生态成熟，能够满足几何处理、多模态模型训练及大规模深度学习计算需求。
